* obrim terminal CMD
* terminal ens posicionem a la carpeta a on tenim l'arxiu .java i el compilem amb l'ordre javac + nomarxiu.java
* una vegada compilat executem amb l'ordre java + nomarxiu.java + passem path del directori que volem veure el contingut.
* drespres d'executarse el programa guarda el resultat en un fitxer amb nomb fileTxt.txt

Ex.
D:\ITAcademyEspecialitzacio\tascaS0105\src\n1exercici1>javac DirectoryList.java
D:\ITAcademyEspecialitzacio\tascaS0105\src\n1exercici1>java DirectoryList.java ../../src
