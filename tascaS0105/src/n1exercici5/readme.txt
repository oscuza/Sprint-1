* obrim terminal CMD
* terminal ens posicionem a la carpeta a on tenim l'arxiu .java i el compilem amb l'ordre javac + nomarxiu.java
* una vegada compilat executem amb l'ordre java + nomarxiu.java + nomNouArxiu i afegira automaticament la terminacio .ser
* una vegada executat llegira l'arxiu, serialitza els objectes de la classe employee i seguidament els serialitza mostrant per consola els objectes creats.

Ex.
D:\ITAcademyEspecialitzacio\tascaS0105\src\n1exercici1>javac DirectoryList.java
D:\ITAcademyEspecialitzacio\tascaS0105\src\n1exercici1>java DirectoryList.java ../../src
